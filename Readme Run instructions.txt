Pre-requisites:
Nodejs installed https://nodejs.org/en/
Nestjs CLI installed https://docs.nestjs.com/
IDE such as VS code or Webstorm

Steps to get running:
1:  Open a terminal with the location set to the directory of the project 
      (most likely the terminal inside your IDE of choice)
2:  Run "npm install" in the terminal
3:  With nest CLI installed run - "nest start"
